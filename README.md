# NOTE.SV

A secure note and password manager based on Bitcoin SV (BSV) Blockchain.

> **Important Notice**: This v1.0 codebase is no longer maintained. New users are recommended to download v2.0 from the official website.

### Development
```bash
npm install

# Desktop version
gulp desktop

# iOS version
gulp ios

# Android version
gulp android
```

### Build

Desktop:
```bash
npm install
gulp buildDesktop
```

iOS:
```bash
npm install
gulp buildIos
```

Android:
```bash
npm install
gulp buildAndroid
```
